## Keybindings:

| Shortcuts | Function |
| ------ | ------ |
| SUPER + K | Show all KeyBindings |
| SUPER + Enter | Terminal |
| SUPER + Q | Kills Active Window |
| SUPER + M | Exit |
| SUPER + E | File Manager |
| SUPER + F | FullScreen |
| SUPER + Shift + F | Toggle Floating |
| SUPER + A | App Menu |
| SUPER + Ctrl + V | Clipboard History |
| SUPER + Ctrl + W | Clear Clipboard History |
| Ctrl + Alt + P | Logout Menu |
| SUPER + L | Lock Screen |
| SUPER + Alt + E | Emoji |
| SUPER + Print | Take Full Screenshot |
| SUPER + Shift + Print | Screenshot selected Area |
| SUPER + Ctrl + Print | Screenshot in 5secs |
| SUPER + Ctrl + Shift + Print | Screenshot in 10secs |
| SUPER + Shift + R | Reload Mango |
| SUPER +  R | Reload Waybar |
| SUPER + T | Title Layout |
| SUPER + V | Vertical Grid |
| SUPER + C | Spiral Layout |
| SUPER + X | Scroller Layout |
| SUPER + N | Switch Layout |
| SUPER + G | Toggle Gaps between windows |
| SUPER + S | Toggles Overview |
| SUPER + Ctrl + [Arrow Keys] | Switch Workspaces |
| SUPER + Shift + [Arrow Keys] | Switch Windows |
| Ctrl + Shift + [Arrow Keys] | Move Windows |
| SUPER + [Arrow Keys] | Move focus to window |
| SUPER + left_mouse | move window |
| SUPER + right_mouse | resize window |
| SUPER + 1 or 2 or... | go to that workspace |
| SUPER + mouse_down | next workspace |
| SUPER + mouse_up | previous workspace |
| SUPER + Shift + 1 or 2 or... | Moves active window to that workspace |
| SUPER + I | Decrease Workspace Size Vertically |
| SUPER + P | Increase Workspace Size Vertically |
