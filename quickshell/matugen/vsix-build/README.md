# Dynamic Base16 HypeShell Theme

A VSCode theme for [HypeMaterialShell](https://github.com/AvengeMedia/HypeMaterialShell).

## How It Works

1. Install this extension
2. Select one of the "Dynamic Base16 HypeShell" themes in VSCode
3. HypeMaterialShell automatically updates the theme files when you change themes.

The theme files are located in your VSCode extensions directory and are updated by matugen when generating new colors.

## Themes

- **Dynamic Base16 HypeShell** - Follows your current light/dark mode
- **Dynamic Base16 HypeShell (Dark)** - Always dark variant
- **Dynamic Base16 HypeShell (Light)** - Always light variant
